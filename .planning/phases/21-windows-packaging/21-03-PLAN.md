<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| build workspace → packaged artifact | The packaging pipeline must not accidentally ship stale or host-specific binaries |
| operator → overwrite upgrade | Human-operated upgrades can leave stale files or partial runtime trees |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-21-08 | T | package assembly output | mitigate | Clean dist directories before packaging and assert the final layout explicitly in smoke checks |
| T-21-09 | I | bundled docs/troubleshooting | mitigate | Document only file locations and failure classes; do not embed secrets or environment-specific credentials |
| T-21-10 | D | overwrite upgrade flow | mitigate | Document stop-before-overwrite and binary-set replacement rules so operators avoid partial upgrades and file-lock deadlocks |
</threat_model>

# Plan 05-04: Flutter UI 组件

> 收藏夹列表、批量选择界面和操作面板的实现

---

## 目标

实现 Flutter UI 层，包括收藏夹列表展示、批量选择交互、操作面板和确认对话框，完成用户界面功能。

## 背景分析

### 已有资产
- flutter_app/lib/widgets/tag_filter_drawer.dart: 侧边栏抽屉组件
- flutter_app/lib/widgets/image_grid.dart: 图片网格组件
- flutter_app/lib/widgets/image_masonry.dart: 瀑布流组件
- CollectionProvider, SelectionProvider (Plan 03 完成)

### 用户决策（必须遵守）
- 入口位置：侧边栏抽屉，上半部分标签筛选，下半部分收藏夹列表
- 批量选择：长按 + AppBar 编辑按钮两种入口
- 选择反馈：蓝色边框 + 勾选图标（Google Photos 风格）
- 操作面板：底部 Bottom Sheet
- 封面策略：自动选择最新添加的图片
- 删除确认：二次确认对话框，显示删除数量

### 需要创建
- 扩展 tag_filter_drawer.dart 支持收藏夹列表
- SelectableImageTile: 可选择的图片卡片
- BatchOperationSheet: 底部操作面板
- DeleteConfirmDialog: 删除确认对话框

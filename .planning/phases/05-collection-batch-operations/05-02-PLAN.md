# Plan 05-02: Service 与 Handler 层

> 收藏夹与批量操作的业务逻辑层和 API 端点实现

---

## 目标

实现收藏夹管理服务、批量操作服务，以及对应的 HTTP Handler，提供完整的后端 API 支持。

## 背景分析

### 已有资产
- CollectionRepository (Plan 01 完成)
- ImageRepository, TagRepository, ImageTagRepository
- Service 层模式：TagGovernanceService 结构
- Handler 层模式：ImageHandler, TagHandler 结构
- Routes 注册模式：SetupRoutes + Dependencies

### 需要创建
- CollectionService: 收藏夹业务逻辑
- BatchService: 批量操作业务逻辑
- CollectionHandler: 收藏夹 API 端点
- BatchHandler: 批量操作 API 端点

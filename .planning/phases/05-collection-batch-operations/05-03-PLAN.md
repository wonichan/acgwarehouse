# Plan 05-03: Flutter Provider 与服务层

> 收藏夹与批量选择的状态管理和服务层实现

---

## 目标

实现 Flutter 端的收藏夹模型、API 服务、状态管理 Provider，为 UI 层提供数据和业务逻辑支持。

## 背景分析

### 已有资产
- flutter_app/lib/providers/image_provider.dart: ImageListProvider 模式
- flutter_app/lib/providers/tag_provider.dart: TagProvider 模式
- flutter_app/lib/services/api_service.dart: ApiService 请求模式
- flutter_app/lib/models/image.dart: ImageModel 模型定义

### 需要创建
- Collection 模型
- CollectionService: API 调用封装
- BatchService: 批量操作 API 封装
- CollectionProvider: 收藏夹状态管理
- SelectionProvider: 批量选择状态管理

       ```go
       type HashService struct {}
       
       // CalculateFileHash 计算 SHA256 文件哈希
       func (s *HashService) CalculateFileHash(path string) (string, error)
       
       // CalculatePHash 计算感知哈希，返回 int64
       func (s *HashService) CalculatePHash(path string) (int64, error)
       
       // HammingDistance 计算两个 int64 的汉明距离
       func (s *HashService) HammingDistance(hash1, hash2 int64) int
       ```

       - 使用 `goimagehash.NewPerceptionHash()` 计算图片 pHash
       - 返回 int64 值
       - 支持 JPG/PNG/WebP/GIF 格式
    
    5. HammingDistance 实现：
       - XOR 两个哈希值
       - 计算置位位数（Popcount）

    2. 在 `internal/repository/schema.go` 添加表定义：
       ```sql
       CREATE TABLE IF NOT EXISTS duplicate_groups (
           id INTEGER PRIMARY KEY AUTOINCREMENT,
           recommended_image_id INTEGER NOT NULL,
           similarity_threshold INTEGER DEFAULT 10,
           created_at TIMESTAMP DEFAULT CURRENT_TIMESTAMP,
           FOREIGN KEY (recommended_image_id) REFERENCES images(id)
       );
       
       CREATE TABLE IF NOT EXISTS duplicate_relations (
           group_id INTEGER NOT NULL,
           image_id INTEGER NOT NULL,
           is_recommended INTEGER DEFAULT 0,
           file_hash TEXT,
           phash_distance INTEGER,
           PRIMARY KEY (group_id, image_id),
           FOREIGN KEY (group_id) REFERENCES duplicate_groups(id) ON DELETE CASCADE,
           FOREIGN KEY (image_id) REFERENCES images(id) ON DELETE CASCADE
       );
       
       CREATE INDEX IF NOT EXISTS idx_duplicate_relations_image ON duplicate_relations(image_id);
       CREATE INDEX IF NOT EXISTS idx_duplicate_relations_file_hash ON duplicate_relations(file_hash);
       ```

  <behavior>

    - Test 2: DetectDuplicates 检测相似图片（pHash 汉明距离 ≤ 阈值）
    - Test 3: DetectDuplicates 使用传递性分组（A~B, B~C → {A,B,C}）
    - Test 4: DetectDuplicates 推荐分辨率最高的图片
    - Test 5: GetDuplicateGroups 返回分页的重复组列表
  </behavior>

    创建 `internal/service/duplicate_service.go`：
    
    ```go
    type DuplicateService struct {
        imageRepo      repository.ImageRepository
        duplicateRepo  repository.DuplicateRepository
        hashService    *HashService
    }
    
    type DetectOptions struct {
        Threshold int  // 汉明距离阈值，默认 10
    }
    
    // DetectDuplicates 执行重复检测
    // 返回检测到的重复组数量
    func (s *DuplicateService) DetectDuplicates(ctx context.Context, opts DetectOptions) (int, error)
    
    // GetDuplicateGroups 获取重复组列表
    func (s *DuplicateService) GetDuplicateGroups(limit, offset int) ([]DuplicateGroupWithImages, error)
    
    // DeleteDuplicateGroup 删除重复组（仅删除记录，不删除图片文件）
    func (s *DuplicateService) DeleteDuplicateGroup(id int64) error
    ```
    
    检测算法实现：
    1. 从数据库加载所有图片
    2. 计算每张图片的文件哈希（如果尚未计算）和 pHash
    3. 按文件哈希分组完全相同的图片
    4. 使用 Union-Find 算法处理传递性分组（pHash 汉明距离 ≤ 阈值）
    5. 每组选择分辨率最高（Width * Height 最大）的图片作为推荐
    6. 保存到 duplicate_groups 和 duplicate_relations 表
    
    使用测试 fixtures 中的重复图片验证算法正确性。

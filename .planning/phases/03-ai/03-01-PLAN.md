  <action>
使用 `golang.org/x/time/rate` 实现 Token Bucket 限流：

1. 添加依赖：
   - 运行 `go get golang.org/x/time/rate`

2. 创建 `RateLimitedClient` 结构体：
   - provider AIProvider - 底层提供商
   - limiter *rate.Limiter - 令牌桶限流器

3. 实现 `NewRateLimitedClient(provider AIProvider, requestsPerMinute int) *RateLimitedClient`：
   - rps = float64(requestsPerMinute) / 60.0
   - burst = 1 (严格限制，不允许突发)
   - 使用 rate.NewLimiter(rate.Limit(rps), 1)

4. 实现 `GenerateTags` 方法：
   - 调用 limiter.Wait(ctx) 等待令牌
   - 等待成功后调用底层 provider

5. 配置集成：
   - 在 AIConfig 中添加 RequestsPerMinute 字段
   - 默认值 60 (1 请求/秒)
  </action>

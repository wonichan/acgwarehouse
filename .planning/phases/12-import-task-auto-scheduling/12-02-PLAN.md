<task type="auto" tdd="true">
  <name>task 2: 实现 AITagAutoScheduler 服务</name>
  <files>
    internal/service/ai_tag_auto_scheduler.go
    internal/service/ai_tag_auto_scheduler_test.go
  </files>
  <behavior>
    - Test 1: ScanAndEnqueue 正确调用 FindImagesWithoutAITags
    - Test 2: ScanAndEnqueue 为每张图片创建平台任务并入队
    - Test 3: ScanAndEnqueue 限制每批最多配置数量
    - Test 4: Start 后定时触发扫描
    - Test 5: Stop 正确停止定时器
    - Test 6: 配置关闭时不执行扫描
  </behavior>
  <action>
    1. 创建 internal/service/ai_tag_auto_scheduler.go：
       ```go
       type AITagAutoScheduler struct {
           imageRepo       repository.ImageRepository
           taskPlatformSvc *TaskPlatformService
           config          *config.Config
           ticker          *time.Ticker
           stopCh          chan struct{}
       }
       
       func NewAITagAutoScheduler(...) *AITagAutoScheduler
       func (s *AITagAutoScheduler) Start(ctx context.Context)
       func (s *AITagAutoScheduler) Stop()
       func (s *AITagAutoScheduler) ScanAndEnqueue(ctx context.Context) (int, error)
       ```
    
    2. 实现 ScanAndEnqueue：
       - 检查配置是否启用 (config.AutoAITag.Enabled)
       - 调用 imageRepo.FindImagesWithoutAITags(limit)
       - 为每张图片构建 TaskPlatformPlanItem
       - 调用 taskPlatformSvc.PlanBatch 创建批次和任务
       - 调用 taskPlatformSvc.QueueTask 入队
       - 返回成功入队数量
    
    3. 实现 Start：
       - 创建 ticker (间隔 = config.AutoAITag.ScanIntervalMin 分钟)
       - 启动 goroutine 监听 ticker 和 stopCh
       - 每次触发调用 ScanAndEnqueue
    
    4. 实现 Stop：
       - 停止 ticker
       - 关闭 stopCh
    
    参考 CONTEXT.md：
    - 扫描频率：每 5 分钟
    - 批量限制：每次最多 100 张
    - 配置开关：auto_ai_tag_on_import
    - 批次来源类型：TaskBatchSourceImportScan
  </action>
